# R404 戰區交通危險係數

## 使用者故事

身為薪資管理人員，
我希望系統能在戰區移動時，
依規定對交通補貼加成危險係數，
以確保戰區交通補貼正確發放。

## 驗收條件

- 給定是否位於戰區 `hazard_zone` 與交通補貼 `transport_allowance`。
- 若位於戰區，則交通補貼 = `transport_allowance * hazard_travel_multiplier`。
- `hazard_travel_multiplier` 來源為 `data/lookup_table.json`。
- `hazard_zone` 來源由 `data/missions.json` 或 `data/units.json` 的欄位提供。
