# R202 國外派遣日支

## 使用者故事

身為薪資管理人員，
我希望系統能依國外派遣國家/城市計算日支，
並按匯率換算成薪資幣別，
以確保海外差旅補貼正確發放。

## 驗收條件

- 給定派遣地點 `country_city` 與派遣天數 `trip_days`。
- 國外派遣日支 = `trip_days * per_diem_international[country_city]`，並乘以 `exchange_rate` 換算。
- `per_diem_international` 與 `exchange_rate` 由規則或參數設定提供。
- `country_city` 與 `trip_days` 來源為 `data/missions.json` 的欄位。
