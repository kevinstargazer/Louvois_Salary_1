# R106 緊急召回

## 使用者故事

身為薪資管理人員，
我希望系統在非勤務時間被緊急召回時，
能發放固定補貼並計入車馬費，
以反映緊急召回的成本。

## 驗收條件

- 給定是否緊急召回 `call_back` 與召回次數 `call_back_count`。
- 緊急召回補貼 = `call_back_count * call_back_flat`。
- 車馬費依交通類規則另計（例如 R401～R405）。
- `call_back_flat` 由規則或參數設定提供。
- `call_back` 與 `call_back_count` 來源為 `data/employees.json` 的欄位。
