# R105 連續勤務疲勞加成

## 使用者故事

身為薪資管理人員，
我希望系統能在連續勤務天數超過門檻後，
對後續勤務津貼加成，
以反映疲勞風險。

## 驗收條件

- 給定連續勤務天數 `streak_days` 與門檻 `streak_threshold`。
- 當 `streak_days >= streak_threshold` 時，後續每一天勤務津貼乘以 `fatigue_multiplier`。
- `streak_threshold` 與 `fatigue_multiplier` 由規則或參數設定提供。
- `streak_days` 來源為 `data/employees.json` 的欄位。
