# R403 私車公用里程

## 使用者故事

身為薪資管理人員，
我希望系統能在員工使用私車執行公務時，
依里程與里程單價計算補貼，並受月上限限制，
以確保里程補貼正確發放。

## 驗收條件

- 給定公務里程 `mileage_km`、里程單價 `mileage_rate` 與月上限 `monthly_mileage_cap`。
- 補貼金額 = `min(mileage_km * mileage_rate, monthly_mileage_cap)`。
- `mileage_rate` 與 `monthly_mileage_cap` 來源為 `data/lookup_table.json`。
- `mileage_km` 來源由 `data/employees.json` 的欄位提供。
