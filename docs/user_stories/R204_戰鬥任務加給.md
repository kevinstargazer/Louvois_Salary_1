# R204 戰鬥任務加給

## 使用者故事

身為薪資管理人員，
我希望系統能在任務類型為戰鬥任務時發放加給，
並可與戰區危險加給疊加但有封頂，
以反映戰鬥風險。

## 驗收條件

- 給定任務類型 `mission_type` 與派遣天數 `trip_days`。
- 若 `mission_type = combat`，則加給 `combat_allowance`。
- 可與 R203 疊加，但總額不得超過 `combat_cap`。
- `combat_allowance` 與 `combat_cap` 由規則或參數設定提供。
- `mission_type` 與 `trip_days` 來源為 `data/missions.json` 的欄位。
