# R104 超時勤務

## 使用者故事

身為薪資管理人員，
我希望系統能針對單日超時勤務的時數計算加給，
以確保超時勤務補償正確發放。

## 驗收條件

- 給定每日工時 `daily_hours` 與標準工時 `daily_standard_hours`。
- 超時時數 = `max(daily_hours - daily_standard_hours, 0)`。
- 超時勤務津貼 = 超時時數 * `overtime_rate`。
- `overtime_rate` 由規則或參數設定提供。
- `daily_hours` 來源為 `data/employees.json` 的欄位。
