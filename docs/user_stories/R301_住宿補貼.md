# R301 住宿補貼

## 使用者故事

身為薪資管理人員，
我希望系統能在未提供軍方宿舍且符合資格時，
依軍階與城市給付住宿補貼，
以確保住宿補貼正確發放。

## 驗收條件

- 給定是否提供宿舍 `barracks_provided`、人員 `rank` 與城市 `city`。
- 若未提供宿舍且符合資格，住宿補貼 = `housing_allowance_table[rank, city]`。
- `housing_allowance_table` 由規則或參數設定提供。
- `barracks_provided` 與 `city` 來源為 `data/units.json` 或 `data/employees.json` 的欄位。
