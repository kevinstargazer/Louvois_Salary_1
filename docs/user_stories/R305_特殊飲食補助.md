# R305 特殊飲食補助

## 使用者故事

身為薪資管理人員，
我希望系統能在符合特殊飲食需求時，
依規定提供特殊飲食補助，
以確保補助正確發放。

## 驗收條件

- 給定是否符合需求 `special_meal_eligible` 與補助金額 `special_meal_allowance`。
- 若符合需求，則給付 `special_meal_allowance`。
- `special_meal_allowance` 來源為 `data/lookup_table.json`。
- `special_meal_eligible` 來源由 `data/employees.json` 的欄位提供。
