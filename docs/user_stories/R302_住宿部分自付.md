# R302 住宿部分自付

## 使用者故事

身為薪資管理人員，
我希望系統能在住宿補貼高於上限時，
只給付到上限並將超過部分視為自付，
以避免超額補貼。

## 驗收條件

- 給定住宿補貼金額 `housing_allowance` 與上限 `housing_cap`。
- 若 `housing_allowance` <= `housing_cap`，則給付 `housing_allowance`。
- 若 `housing_allowance` > `housing_cap`，則給付 `housing_cap`，超過部分不入薪。
- `housing_allowance` 來源為 R301 計算結果；`housing_cap` 來源為 `data/lookup_table.json`。
