# R108 訓練日津貼

## 使用者故事

身為薪資管理人員，
我希望系統能針對訓練日給付每日津貼，
若訓練在戰區則改用戰區規則，
以確保訓練津貼與戰區津貼不重複。

## 驗收條件

- 給定訓練日數 `training_days` 與訓練是否在戰區 `training_in_hazard_zone`。
- 訓練在非戰區時，訓練日津貼 = `training_days * training_day_allowance`。
- 訓練在戰區時，訓練日津貼不給付，改用戰區相關規則計算（互斥/優先）。
- `training_day_allowance` 由規則或參數設定提供。
- `training_days` 來源為 `data/employees.json` 的欄位；是否戰區由 `data/missions.json` 的 `hazard_zone` 推導。
