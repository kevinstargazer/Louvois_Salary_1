# R205 海外艦艇勤務

## 使用者故事

身為薪資管理人員，
我希望系統能在海上連續勤務達門檻天數時給付額外加給，
以反映海外艦艇勤務的特殊性。

## 驗收條件

- 給定海上連續勤務天數 `sea_days` 與門檻 `sea_days_threshold`。
- 若 `sea_days >= sea_days_threshold`，則額外加給 `sea_days * sea_duty_bonus_per_day`。
- `sea_days_threshold` 與 `sea_duty_bonus_per_day` 由規則或參數設定提供。
- `sea_days` 來源為 `data/missions.json` 的欄位。
